# DBMS Salary Management System

## Overview

The DBMS Salary Management System is a web-based application designed to manage employee salary records efficiently. The system automates salary calculations, employee record management, and payroll-related operations, reducing manual effort and improving accuracy.

This project demonstrates the use of database management concepts, CRUD operations, and web technologies to create a centralized salary management solution.

---

## Features

### Employee Management

* Add new employee records
* Update employee details
* Delete employee records
* View employee information

### Salary Management

* Generate salary records
* Calculate monthly salary
* Manage allowances and deductions
* View salary history

### Database Operations

* Store employee and salary information securely
* Perform Create, Read, Update, and Delete (CRUD) operations
* Maintain data consistency and integrity

### User Interface

* Simple and user-friendly dashboard
* Easy navigation for administrators
* Responsive design for improved usability

## Database Design

### Employee Table

| Field       | Description                |
| ----------- | -------------------------- |
| Employee_ID | Unique employee identifier |
| Name        | Employee name              |
| Department  | Employee department        |
| Designation | Job role                   |
| Contact     | Contact details            |

### Salary Table

| Field        | Description              |
| ------------ | ------------------------ |
| Salary_ID    | Unique salary identifier |
| Employee_ID  | Employee reference       |
| Basic_Salary | Base salary              |
| Allowances   | Additional benefits      |
| Deductions   | Salary deductions        |
| Net_Salary   | Final calculated salary  |

---

## Project Workflow

1. Administrator logs into the system.
2. Employee information is added and stored in the database.
3. Salary details are entered for each employee.
4. The system calculates net salary based on allowances and deductions.
5. Salary records are stored and can be viewed or updated when required.
6. Reports can be generated for payroll management.
